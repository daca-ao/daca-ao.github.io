ArrayList 源码浅析

Posted on 2021-06-07

ArrayList 是 List 接口最常用的通用实现。

  • 底层由一个 Object 数组维护
  • 构造时会设置一个初始的容量，元素增加时会自动扩容
  • 必要时：如元素为大对象，可手动缩容
  • 线程不安全

构造方法

动态扩容

ArrayList 添加元素前会确保容量足够，如不足则会进行扩容

  • 初始容量 DEFAULT_CAPACITY = 10
  • 添加前调用 ensureCapacityInternal(size + 1)，size 为当前元素个数

抛开对于溢出情况的限制逻辑，扩容关键语句：

  • 容量不足时，新容量为旧的 1.5 倍
  • 位运算容量计算效率更高
  • 新容量为旧的 1.5 倍加 1

扩容后续：

  • 确定 newCapacity 后会申请一个长度为 newCapacity 的 Object 数组
  • 接着调用 Arrays.copyOf() 将原数组复制过去

扩容是通过将整个数组拷贝的方式完成：对于大对象数组需考虑性能问题，提前规划容量，降低扩容频率。
