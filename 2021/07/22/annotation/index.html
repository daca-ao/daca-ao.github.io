Java 注解（Annotation）

Posted on 2021-07-22 Views:

从 Java 5 开始，官方开始引入注释机制，程序员们也可以通过自定义注解实现各种需求。

Java 注解是通过反射获取注解内容的；在编译器生成类文件时，注解可被嵌入到字节码中。

  • JVM 可保留注解的内容，运行时会获得注解内容
  • 类、方法、变量、参数、包都可被标注
  • 支持自定义注解
